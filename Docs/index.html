Introduction

Creating better quality art using machine learning techniques is imperative for reaching human-like capabilities, as well as opens up a new spectrum of possibilities. And with the advancement of computer hardware as well as the proliferation of deep learning, deep learning is right now being used to create art. How to get started ?

  1. Fork this repository.
  2. Clone the forked repository.
    git clone https://github.com/your-github-username/DeepPixel
  3. Navigate to the project directory.
    cd DeepPixel
  4. Create a new branch.
    git checkout -b your_branch_name
  5. Make changes in source code.
  6. Commit your changes.
    git add .
    git commit -m "your_commit_message"
  7. Push your local branch to the remote repository.
    git push -u origin "your_branch_name"
  8. Create a Pull Request!

    9. Congratulations! You have just made your contribution to DeepPixel project.

Current Issues

1. Face Recognition

  • The task is to input an image and output the image with faces detected (boundary boxes are drawn)
  • Count the number of faces detected
  • Real-Time Face Detection
  • A pre-trained model(like MobileNet) that takes in a custom dataset of faces and recognizes the faces with labels in other images
  • Get the gender and age

2. Real Time Pose Detection

  • The function should access a webcam or feed any video and estimate the position of a person based upon their key joint positions
  • For reference: https://medium.com/tensorflow/real-time-human-pose-estimation-in-the-browser-with-tensorflow-js-7dd0bc881cd5

3. Super Resolution

  • The task is to take a small, low-resolution image and output a high-resolution image without the formation of additional artifacts

4.Image Similarity

  • Compare two input images and return a value that tells how visually similar the given images are.
  • The lower the score, the more contextually similar the two images are with a score of '0' being identical.

4.Rock,paper and Scissor Game Implmentation

  • The task is to to identify the three hand gestures i.e Rock , Paper and Scissor and create a game .
  • Implementation : Neural Netwrok/MLP/logistic Regression ,OpenCV threshold can be used .

5.Facial Expression Recognition

  • The task is to to take in an image of a face as input and predict the expression
  • The experssion can be happy, sad, angry, scared etc

6.Image Captioning

  • The task is to to take in an image and provide a proper caption to it.
